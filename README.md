# AwardHunter India ✈

Find the cheapest award flight tickets using Indian credit card points.

**Features:**
- 100+ international airports
- 14 Indian credit card programs (Axis Atlas, HDFC Infinia, Amex Platinum Travel, etc.)
- 13 airline loyalty programs
- Data from PointsYeah, AwardFares & Seats.aero
- Calendar view showing cheapest dates
- Price alerts
- Deep links to airline booking pages
- Transit visa filtering for Indian passports
- Business, First & Premium Economy

## Deploy

### Vercel (Recommended — Free)
```bash
npm install
npx vercel
```

### Netlify (Free)
```bash
npm install
npm run build
npx netlify deploy --prod --dir=dist
```

### Local Development
```bash
npm install
npm run dev
```

## Tech Stack
- React 18 + Vite
- Zero external UI dependencies
- Single-file dashboard component
